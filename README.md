# Neural Network MultiLang

A collaborative project to implement neural networks from scratch across multiple programming languages. 

## The Concept

This project started as `NeuroGo` (a neural network built purely in Go) and has evolved into a multi-language exploration of deep learning fundamentals. By rewriting the exact same neural network architectures in different languages, we aim to:
- Compare the specific syntax, paradigms, and design patterns of different languages.
- Analyze the performance differences (execution speed, memory usage) between compiled and interpreted languages.
- Create a collaborative learning environment for exploring AI concepts from the ground up without relying on heavy frameworks.

## Supported Languages

Currently, the implementations are divided into the following directories:

* **`/go`** - The foundational implementation written in Go. Features Feedforward and Backpropagation.
* **`/python`** - Python implementation (In progress).
* **`/java`** - Java implementation (In progress).
* **`/cpp`** - C++ implementation (In progress).

## Shared Resources

Any datasets or universal documentation relevant across all implementations are stored in the `/data` folder at the root level.

## Getting Started

To run or contribute to a specific language implementation, navigate into the respective folder and follow the instructions provided within (if any) or look at the source code.

For example, to run the Go implementation:

```bash
cd go
go run main.go
```
